■こんなことができます

Windows Media Player（Ver. 6.4以上）でSony ICレコーダー用ファイル(msv形式、
dvf形式)を再生させる事ができます。

■使用方法

ウインドウズのエクスプローラ上でmsv/dvfファイルをダブルクリックします。
Windows Media Playerが起動し、再生が始まります。

Windows Media Playerからmsv/dvfファイルを開くこともできます。

■アンインストール

[スタート]ボタンをクリックし、[プログラム]にカーソルを合わせ、[Sony 
Player Tool for WMP]から[アンインストール]をクリックします。
アンインストーラが起動します。画面の指示に従ってアンインストールを行って
ください。

＜必要なシステム構成＞
  このソフトウェアを使うためには、次のようなハードウェア、ソフトウェアが
必要です。

□以下の性能を満たしたIBM PC/ATおよびその互換機
・CPU : 266MHz以上のPentium IIプロセッサーもしくは同等の性能を有するプロ
	セッサー（NEC PC-98シリーズとその互換機、また自作PCでは動作保証い
	たしません。Macintoshには対応していません。）
・RAM容量 : 64Mバイト以上
・ハードディスクの空き容量 : 5Mバイト以上
・サウンドボード : Sound Blaster 16互換
・ディスプレイ : ハイカラー（16ビットカラー）以上、640x480ドット以上
・OS : Microsoft Windows XP Home Edition/Professional
                 Windows 2000 Professional
                 Windows Millennium Edition
                 Windows 98/98 Second Edition標準インストール（日本語版）
      （Windows 95またはWindows NTには対応していません。）
      （推奨環境すべてのパソコンについて動作を保証するものではありません。）
・Windows Media Player : Ver. 6.4以上
	お使いのパソコンがWindows Media Playerが必要なシステム構成（動作
	環境）を満たしている必要があります。
