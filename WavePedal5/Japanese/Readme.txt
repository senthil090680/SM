【修正履歴(Ver.2.01)】
 1.ADPCM形式追加時の不具合修正
  msv(ADPCM)ファイル形式の用件をメモリースティックへADPCM形式で追加する時、
  通常よりも時間がかかってしまう場合がある問題を修正しました。
 2.音声フォーマット変換時の不具合修正
  LPの用件を音声フォーマット変換する時、msv(LPEC)ファイル形式からdvfファイ
  ル形式への変換、またはその逆の変換を行うと、ユーザー名・タイトルが引き継
  がれない問題を修正しました。
 3.音声フォーマット変換/結合時の不具合修正
  特定のdvfファイルに対して音声フォーマット変換を行うと「対応していない形
  式のため、変換できませんでした。」と表示されてしまい変換できなかったり、
  結合を行うとMemory Stick Voice Editorが応答なしになってしまう場合がある
  問題を修正しました。
 4.MP3ファイル追加時の動作修正
  メモリースティックへ追加できない形式のMP3ファイルを追加しようとした場合、
  すぐにエラーメッセージが表示されるように修正しました。


＜MP3ファイルの表示/操作のご説明＞
Memory Stick Voice EditorではMP3ファイルを以下の方法で表示/操作することが
できます。

■ MP3ファイルの表示
メニューの[表示]→[MP3ファイルの表示]をチェック状態にすると、PC側用件表
示部に拡張子'.mp3'のファイルが表示されます。このとき、リスト項目の「ユー
ザー名」「タイトル」にはMP3ファイルのアーティスト名、トラック名に相当す
る情報が表示されます。

■ MP3ファイルの再生
PC側用件表示部に表示されたMP3ファイルを選択してパソコン上で再生させるこ
とができます。また、早送り、早戻し、DPC、ABリピートなども行えます。

！ご注意
・MP3の形式によっては再生できないものもあります。

■ MP3ファイルのメモリースティックへの追加
MP3ファイルに対して追加操作を行うと、メモリースティック上にはメモリース
ティックボイスフォーマット（[オプション]→[コーデック]で指定したフォーマ
ット）に変換された用件が追加されます。
　追加された用件には追加元MP3ファイルのユーザー名（msv(LPEC)ファイル形式
の場合のみ）、タイトルが引き継がれます。追加された用件はそのままメモリー
スティックICレコーダーで再生させることができます。

！ご注意
・MP3形式のまま追加することはできません。
・追加できるMP3ファイルは44.1kHzのステレオ、または22/16/11/8kHzのモノラル
  です。また、再生できないMP3ファイルを追加することはできません。
・アーティスト名の長さが半角で20文字を超える場合は、ユーザー名には半角で
  20文字までが引き継がれます。

■ MP3ファイルのメール送信
MP3ファイルをメールに添付して送信させることができます。詳しい操作方法は
ヘルプの「音声メールを送信する」の項目をご覧ください。

■ MP3ファイルでは行えない操作
MP3ファイルでは次の操作を行うことができません。
・ユーザー名/用件タイトルの変更
・重要マークの設定
・ブックマーク設定
・分割/結合
・音声フォーマット変換
・音声認識


＜ご注意＞
1.インストール時のご注意
  インストール先のフォルダ名には全角文字を使わないでください。Memory Stick 
  Voice Editorが起動できなくなります。この場合は一度Memory Stick Voice 
  Editorをアンインストールし、再度インストールを行ってください。

  Windows(R) 2000 Professional上でインストールを行う場合、必ずユーザー名 
  Administrator でログオンした後に行ってください。

  Windows(R) XP Professional / XP Home Edition上でインストールを行う場合、
  必ず「コンピュータの管理者*」に所属するユーザー名でログオンした後に行っ
  てください。
  *ユーザー名が「コンピュータの管理者」に所属しているかの確認は、Windowsの
  [コントロールパネル]-[ユーザー アカウント]を開き、表示されるユーザー名の
  下の部分をご覧ください。

2.システムサスペンド/システムハイバネーションモードについてのご注意
  Memory Stick Voice Editor を起動している状態でシステムサスペンド/システム
  ハイバネーションモードへ移行しないでください。自動的に移行するような設定は
  避けてください。

3.メール送信機能利用上のご注意
  MAPI(Microsoft Messaging Application Programming Interface)準拠の以下の
  メールソフトウェアをお使いになれます。（別売り）
  「通常使用するメールソフト」に設定することにより、Memory Stick Voice 
  Editor から起動することができます。詳しくはヘルプの「音声メールを送信する」
  をご覧ください。
	Microsoft Outlook Express 5.0/5.5/6.0（日本語版）
	Microsoft Outlook 2000/2002（日本語版）
	Eudora Ver.4.2-J/4.3-J（ペイドモード）/5.0-J（製品版）
  メールソフトウェアをインストールするには、お使いのパソコンが必要なシステ
  ム構成（動作環境）を満たしている必要があります。

  Eudoraを使用してメール送信を行う場合、以下のいずれかの設定を行ってください。
  ・EudoraのMAPIの設定を「常時」にする。
    （設定方法はEudoraの取扱説明書をご覧ください。）
  または
  ・インターネットエクスプローラのバージョンを5.0以上にする。

  受信側のメールソフトが実行形式のファイルを受信しない設定等になっている場合、
  プレーヤーツールのセットアップファイルを開けない事があります。

4.メールの添付ファイルサイズに関するご注意
  インターネットプロバイダによっては、添付ファイルサイズの制限がある場合があ
  ります。送信する前に、必ず合計添付ファイルサイズを確認してください。

5.ファイルフォーマット変換機能のご注意
  ファイルフォーマット変換機能は録音時間に比例して処理時間が長くなります。  
  ご注意ください。

6.メモリースティックICレコーダーICD-MS1/MS2およびMemory Stick Voice Editor 
  Ver.1.0/1.1/1.2をお使いのお客様へのご案内

  ICD-MS500にて録音された用件はmsv(LPEC)ファイル形式のため、ICD-MS1/MS2及び
  Memory Stick Voice Editor Ver.1.0/1.1/1.2では再生する事ができません。再生
  するためには、Memory Stick Voice Editor Ver.2にて一度用件をパソコンに保存
  し、メモリースティックへmsv(ADPCM)ファイル形式で再度追加してください。msv
  (ADPCM)ファイル形式で追加させる方法はヘルプの「保存・追加ファイル形式を設
  定する」をご覧ください。

  msv(ADPCM)ファイル形式は、以下の機能を使用することができません。あらかじめ
  ご了承ください。
  −ユーザー名の変更
  −分割・結合
  −ブックマーク
  −重要度（メモリースティック上での設定は可能です。）

7.USB接続ICレコーダーICD-BPシリーズ用アプリケーションDigital Voice Editor 
  Ver.1.0/1.01/1.1を使用する場合のご注意
  
  Digital Voice Editor Ver.1.0/1.01/1.1はmsv(LPEC)ファイル形式に対応していな
  いため、msv(LPEC)ファイル形式を再生/追加させる事ができません。この場合は
  Memory Stick Voice Editorでdvfファイル形式に変換してから、Digital Voice 
  Editorで使用してください。

8.USB接続ICレコーダーICD-BPシリーズ用アプリケーションDigital Voice Editor 
  Ver.1.0/1.01/1.1をインストールする場合のご注意

  お使いのパソコンにDigital Voice Editor Ver.1.0/1.01/1.1をインストールする
  と、msvファイルのアイコンの色がすべて青になる場合があります。この場合もう
  一度Memory Stick Voice Editor Ver.2のインストールを行うとアイコンが正常に
  表示されるようになります。

9.音声認識に関してのご注意
  お使いのパソコンにDragonSpeech Select/Professional Ver. 4.05/5.0がインス
  トールされている必要があります。（別売り）
  DragonSpeech Partnerには対応していません。
  DragonSpeech Select Ver. 3.6には対応していません。
  DragonSpeechをインストールするには、お使いのパソコンが DragonSpeech が必
  要なシステム構成（動作環境）を満たしている必要があります。

10.音量調整についてのご注意
  お使いのパソコンによっては音量調整が正常に動作しないことがあります。
  その場合は、ウインドウズ標準のボリュームコントロールウインドウ上でボリュ
  ーム調整を行ってください。
  [スタート]メニューから[プログラム]→[アクセサリ]→[エンターテイメント]
  →[ボリュームコントロール]をクリックしてください。ボリュームコントロール
  のウインドウが表示されます。ここでボリューム調整を行ってください。
  ミュートを解除しても音が出ない場合は、ウインドウズ標準のボリュームコント
  ロールのウインドウで、一度ミュートにチェックを入れ、その後ミュートを解除
  してください。音が出るようになります。
